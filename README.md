Інструкція для Discord-бота “Void Sentinel”

Встановлення Python

Переконайтеся, що на комп’ютері встановлено Python 3.10 або вище.

Перевірте командою:

python --version


Клонування репозиторію

Скопіюйте репозиторій з GitHub:

git clone https://github.com/Kraken-Legionary/Void-Sentinel.git
cd Void-Sentinel


Встановлення залежностей

Встановіть необхідні бібліотеки через pip:

pip install -r requirements.txt


(Якщо файлу requirements.txt немає, можна вказати основні бібліотеки: discord.py, python-dotenv.)

Налаштування токена бота

Створіть файл .env у кореневій директорії.

Додайте рядок:

DISCORD_TOKEN=ВАШ_ТОКЕН_БОТА


Токен можна отримати у Discord Developer Portal
.

Запуск бота

Виконайте команду:

python main.py


Бот підключиться до сервера і буде готовий до роботи.

Перевірка роботи

Відправте базову команду, наприклад !ping або /ping, щоб переконатися, що бот відповідає.

Перевірте роботу адміністративних та інтерактивних функцій у тестовому середовищі.
